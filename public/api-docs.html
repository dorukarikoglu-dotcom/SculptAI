SculptAI API v1

Base URL: https://app.sculptai.health/api/v1

Authentication

All requests require a Bearer token in the Authorization header:

Authorization: Bearer YOUR_API_KEY_HERE

API keys are created in the Doctor Panel under Ayarlar > API & Webhooks. Keys starting with sk_live_ access production data; sk_test_ keys access test data.

API keys are shown only once at creation. Store them securely.

Rate Limits

WindowLimit
Per minute100 requests
Per hour1,000 requests
Per day10,000 requests

When exceeded, the API returns 429 Too Many Requests with a retry_after_seconds field.

Endpoints

POST/patients

Create a new patient record

Request Body

FieldTypeRequiredDescription
namestringYesPatient name
agenumberYesPatient age
procedurestringYesProcedure name
genderstringNoGender
phonestringNoPhone number
form_dataobjectNoAdditional form answers
utm_sourcestringNoUTM source tracking
languagestringNo"tr" or "en"
curl -X POST https://app.sculptai.health/api/v1/patients \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"name":"Ayse Yilmaz","age":32,"procedure":"Burun Estetigi"}'

GET/patients

List patients with pagination and filtering

Query Parameters

ParamTypeDescription
pagenumberPage number (default: 1)
limitnumberPer page (default: 20, max: 100)
segmentstringFilter by segment
from_dateISO dateFilter from date
to_dateISO dateFilter to date
outcomestring"no_appointment" or "had_procedure"
curl https://app.sculptai.health/api/v1/patients?page=1&limit=10&segment=Randevuya%20Haz%C4%B1r \
  -H "Authorization: Bearer YOUR_API_KEY"

GET/patients/:id

Get patient details including observation and outcomes

curl https://app.sculptai.health/api/v1/patients/PATIENT_ID \
  -H "Authorization: Bearer YOUR_API_KEY"

PATCH/patients/:id

Update patient (allowed: phone, notes, no_appointment)

curl -X PATCH https://app.sculptai.health/api/v1/patients/PATIENT_ID \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"notes":"VIP hasta","phone":"05551234567"}'

DELETE/patients/:id

Delete a patient record

GET/patients/:id/observation

Webhooks

When events occur, SculptAI sends a POST request to your registered URL with:

HeaderDescription
X-SculptAI-SignatureHMAC-SHA256 signature (if secret provided)
X-SculptAI-TimestampUnix timestamp
X-SculptAI-EventEvent type
X-SculptAI-DeliveryUnique delivery ID

Verification

const expected = hmac_sha256(secret, timestamp + "." + raw_body);
if (expected === signature) { /* valid */ }

JavaScript Example

const API = "https://app.sculptai.health/api/v1";
const KEY = "YOUR_API_KEYxxxxx";

// Create patient
const res = await fetch(`${API}/patients`, {
  method: "POST",
  headers: {
    "Authorization": `Bearer ${KEY}`,
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    name: "Ayse Yilmaz",
    age: 32,
    procedure: "Burun Estetigi",
  }),
});
const { patient } = await res.json();
console.log(patient.id);

// List patients
const list = await fetch(`${API}/patients?page=1&limit=10`, {
  headers: { "Authorization": `Bearer ${KEY}` },
});
const { patients, pagination } = await list.json();

// Get analytics
const stats = await fetch(`${API}/clinic/analytics`, {
  headers: { "Authorization": `Bearer ${KEY}` },
});
const { analytics } = await stats.json();
